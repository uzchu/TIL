# 챗봇 실습

마크로프 체인기능 활용한 챗봇 만들기 실습



- chatbot.py 문서 구동방법

```
콘다 관리자권한으로 실행해서 cgi-bin폴더가 있는 폴더에서 아래 코드 침
(base) c:\py_data>python -m http.server --cgi 8000

크롬으로가서 localhost:8000/cgi-bin/chatbot.py
```
