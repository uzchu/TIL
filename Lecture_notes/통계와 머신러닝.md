## 머신러닝

machine learning

- machine(=function, algorithm, model)
- learning(=change) : 학습의 궁극의 목적 '변화' => 변경의 대상은 파라메타

예측과 분류

예측

- 회귀식 : y(output) = w * x(input) + b
- 일반적으로 파라메타 갯수가 높아지면 설명력이 높아짐
- 예측값과 실제값의 차이가 loss, 모델생성은 이를 최소화하는 함수를 찾는 것
