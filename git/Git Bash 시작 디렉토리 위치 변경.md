# Git Bash 시작 디렉토리 위치 변경

Git Bash에서 마우스 오른쪽 키 > 속성을 누른다.

“바로 가기” 탭의 “시작 위치(S):”를 보면 “%HOMEDRIVE%%HOMEPATH%”로 되어 있을텐데, 이를 시작하고자 하는 위치로 바꾼다.

그 위의 “대상(T):” 을 보면 마지막에 git-bash.exe의 뒤에 “–cd-to-home” 이 붙어 있을텐데 이를 지운다.
